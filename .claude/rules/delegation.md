# 委譲ルール

オーケストレーター自身はコードを書かない。コード実装・コード編集はすべてサブエージェントに委譲する。
（指示書・ループプロンプトなどのドキュメントファイルの操作はオーケストレーター自身が行ってよい）

| 実装対象           | 委譲先               |
| ------------------ | -------------------- |
| バックエンド実装   | `sample-api-agent`   |
| フロントエンド実装 | `sample-front-agent` |
| E2E テスト実装     | `sample-front-agent`（`/e2e-gen` 経由のみ） |
