# サービス概要

spec-to-dev-workflow は、仕様（spec）から開発（dev）へのワークフローを管理するサービスです。

バックエンド API（`sample-api`）とフロントエンド（`sample-front`）の2つのシステムで構成されており、仕様書を起点に実装・テストまでを一貫して管理します。

## 機能一覧

| 機能                        | 概要                                                  |
| --------------------------- | ----------------------------------------------------- |
| [Hello](./hello/README.md)  | サービスの疎通確認。挨拶メッセージを取得して表示する  |
