# 用語集

プロジェクト全体で使用するドメイン用語の正規定義。

`specs/` 配下の仕様書を書く際は、ここに定義された用語を優先して使用する。
新しいドメイン用語が登場したら、この用語集に追記する。

---

## ドメイン用語

| 用語 | 定義 | 補足・使用例 |
|---|---|---|
| グループ | ユーザーをまとめる単位 | API では `group` |
| メンバー数 | グループに所属するユーザーの人数 | `member_count` |
| 検索 | キーワードによる一覧の絞り込み操作 | 「絞り込み」「フィルター」は使わない |
| ページネーション | ページ単位で一覧を分割するナビゲーション。`page` / `limit` パラメータで制御する | Previous / Next ボタン、ページ情報 |
