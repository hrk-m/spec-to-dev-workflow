# select-all-members — ヘッダーチェックボックスで全メンバーを一括選択

## 概要

メンバー一覧テーブルのヘッダー左端に「全選択」チェックボックスを追加し、クリック 1 回で表示中の全メンバーを選択または全解除できる。一部だけ選択されている場合はヘッダーチェックボックスが indeterminate（半選択）状態になり、クリックすると全選択に切り替わる。

---

## 処理フロー（正常系）

```
ユーザーがヘッダーチェックボックスをクリックする
  │
  ├─ 全未選択の場合 → 全メンバーが選択される（削除ボタンが enabled に変わる）
  ├─ indeterminate（一部選択）の場合 → 全メンバーが選択される
  └─ 全選択の場合 → 全メンバーの選択が解除される（削除ボタンが disabled に戻る）
```

---

## ヘッダーチェックボックスの状態派生

| 条件                             | ヘッダーチェックボックスの状態 |
| -------------------------------- | ------------------------------ |
| メンバー 0 件                    | disabled                       |
| メンバー 1 件以上 かつ 全未選択  | 未チェック                     |
| 0 < 選択数 < 全体数              | indeterminate（半選択）        |
| 選択数 = 全体数（かつ 1 件以上） | チェック済み                   |
| 削除成功後                       | 全解除（リセット）             |

---

## 処理フロー（異常系）

```
メンバーが 0 件の場合
  │
  └─ ヘッダーチェックボックスが disabled になり、クリックできない

削除 API 失敗の場合
  │
  └─ 選択状態を維持したまま AlertDialog 内にエラーを表示する（ヘッダー状態は変化なし）
```

---

## 使用コンポーネント・状態

| 要素                | 種別           | 役割                                                                                                 |
| ------------------- | -------------- | ---------------------------------------------------------------------------------------------------- |
| `MemberList`        | コンポーネント | ヘッダーチェックボックスと全選択ロジック・indeterminate の DOM 反映を担う                            |
| `headerCheckboxRef` | ref            | ヘッダー `<input type="checkbox">` への参照。`useEffect` で `indeterminate` プロパティを直接設定する |
| `isAllSelected`     | 派生 state     | `members.length > 0 && selectedIds.size === members.length` の場合 true                              |
| `isSomeSelected`    | 派生 state     | `selectedIds.size > 0 && selectedIds.size < members.length` の場合 true                              |
| `handleSelectAll`   | イベント関数   | 全選択 / 全解除を切り替える。`isAllSelected` が true のとき全解除、それ以外は全選択                  |
| `selectedIds`       | state          | チェックされたメンバー ID の集合（`Set<number>`）。既存の削除フローと共有                            |

---

## 確認観点

```
- [ ] ヘッダーチェックボックスをクリックすると全メンバーが選択される
- [ ] 全選択状態でヘッダーチェックボックスをクリックすると全メンバーの選択が解除される
- [ ] 一部のメンバーのみ選択した状態ではヘッダーチェックボックスが indeterminate 表示になる
- [ ] indeterminate 状態でヘッダーチェックボックスをクリックすると全選択になる
- [ ] メンバーが 0 件のときヘッダーチェックボックスが disabled になる
- [ ] 全選択後に削除が成功するとヘッダーチェックボックスが未チェック・disabled にリセットされる
- [ ] ヘッダーチェックボックスの変化に合わせて「削除」ボタンの enabled / disabled が連動する
```
