# グループ管理機能

## 概要

「グループ管理」機能は、システム内のグループ情報を管理する機能です。

グループの一覧を検索・閲覧でき、ページネーションで大量のデータにも対応しています。

## 目的

- グループの一覧を検索・閲覧できるようにする
- ページネーションにより大量データの効率的な閲覧を可能にする

## 画面フロー

1. ユーザーがトップページを開く
2. グループ一覧が自動的に読み込まれる（読み込み中はスケルトン表示）
3. グループの名称・メンバー数・説明が一覧表示される
4. 検索ボックスにキーワードを入力すると、名前・説明で絞り込める
5. 「Previous」「Next」ボタンでページを切り替えられる

## データモデル

| 項目       | 説明                       | 例                       |
| ---------- | -------------------------- | ------------------------ |
| ID         | グループの識別子           | `1`                      |
| 名称       | グループの名前             | `"Engineering"`          |
| 説明       | グループの説明文           | `"エンジニアリングチーム"` |
| メンバー数 | グループに所属する人数     | `12`                     |
