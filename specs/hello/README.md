# Hello 機能

## 概要

「Hello」機能は、サービスが正常に動作しているかを確認するためのシンプルな疎通確認機能です。

画面を開くと、バックエンドから挨拶メッセージを取得して表示します。

## 目的

- バックエンドとフロントエンドの連携が正しく機能しているかを確認する
- 新しい機能を追加する際の実装パターンの参照例として使用する

## 画面フロー

1. ユーザーがトップページを開く
2. 画面に「loading...」と表示される
3. バックエンドからメッセージを取得する
4. メッセージが取得できた場合 → 取得したメッセージを表示する
5. メッセージが取得できなかった場合 → エラー内容をサーモンピンク色で表示する

## データモデル

| 項目       | 説明           | 例         |
| ---------- | -------------- | ---------- |
| メッセージ | 挨拶の文字列   | `"hello"`  |
